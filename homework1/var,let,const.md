1. Обьяснить своими словами разницу между обьявлением переменных через var, let и const. 
    >Разница между var и let в области видимоти, переменная var которая объявлена внутри фигурных скобо будет доступный глобально, а переменная которая будет обьявлена через let будет ограниченна скобками. Видимость const аналогична let, const не позволяет переписывать некую ссылку на определенную область памяти, где распологаются данные, невозожно перезаписывать
